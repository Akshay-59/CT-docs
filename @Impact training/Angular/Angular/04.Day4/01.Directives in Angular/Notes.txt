Directives in Angular
~~~~~~~~~~~~~~~~

Types of Directives
		 Components directives
		 Structural directives
		 Attribute directives


 Components directives
 ~~~~~~~~~~~~~~~~~~~~
	--> Every component in angular is consider as directive.
	--> These comes under component directive.	
 
	Eg:		<app-emps>, <app-root>, ...
 

 Structural directives
 ~~~~~~~~~~~~~~~~~
	 --> Responsible for HTML layout.
	--> Angular manipulates the DOM with structural directives.
	--> Change the DOM by adding and removing DOM elements.
	Eg:  *ngFor, *ngIf, *ngSwitchCase etc…


Attribute Directives
~~~~~~~~~~~~~~~~~
--> Change the appearance or behavior of an element.
--> Attribute directives are used as attributes of elements (tags).
		Eg: ngModel , ngClass, routerLink
	




Built-in directives:	
~~~~~~~~~~~~~~
	*ngFor
	*ngIf
	 ngSwitch
	*ngSwitchCase
	*ngSwitchDefault	
	
	ngModel
	ngClass
	ngStyle
		
		
		
		
Usage  ngStyle  and ngClass
~~~~~~~~~~~~~~~~~~~~~~~~

	<tag  [ngStyle]="{ 'color' : 'red' }"> </tag>
	
	Syntax:	<tag  [ngClass]="{ 'cssClassName' : boolean }"> </tag>
	
	Eg:	<tag  [ngClass]="{ bg1 : true }"> </tag>
			<tag  [ngClass]="{ 'btn btn-primary' : true }"> </tag>
									
			
	*ngFor="let item of  arrayName; let i = index; let x = even; let y = odd;"		
	
	
	index	---		even		---		odd
	--------------------------------------------
	0					true				false
	1					false			true
	2					true				false
	3					false			true
	4					true				false
	5					false			true
	
	
	
	To apply css classes to alternate items:
	
	<tr  *ngFor="let item of  arrayName; let i = index; let x = even; let y = odd;"		
			[ngClass]="{ bg1 : x, bg2 :  y }"> 
			.............
	</tr>
